- Architected a **Java (Maven) automation &amp; execution platform** powering **350+ production bots** across web, desktop, API, and OCR surfaces — standardizing config, execution, and reporting.
- Led a **zero-downtime migration of 1,200+ modules off JUnit** onto an in-house Reflection-based runner, cutting compile overhead and dramatically simplifying maintenance.
- Owned CI/CD at scale: **1,200+ Jenkins jobs** orchestrated via a custom **Groovy Shared Library** for cross-platform (Unix/Windows) pipelines on GitLab CI/CD.
- Designed custom **agentic-AI test agents** (GPT/Gemini-driven) embedded into automation workflows to auto-generate machine-readable test data; drove design &amp; merge-request reviews and team knowledge-sharing.

- Revamped the **HCM Report Generator** — reduced report load time and shipped new reporting features.
